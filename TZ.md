# Техническое задание на проект: Веб-сервис расчета ROI солнечных электростанций для частных домовладельцев с агрегацией актуальных рыночных цен через API поставщиков

## 1. Цель проекта
Веб-сервис для расчета окупаемости солнечных панелей для частных домовладельцев. Позволяет определить срок возврата инвестиций на основе актуальных рыночных цен, региональных тарифов на электроэнергию и солнечной инсоляции.

## 2. Роли пользователей
- **Гость**: Может выполнять расчеты без сохранения результатов
- **Авторизованный пользователь**: Может сохранять расчеты, просматривать историю, получать персонализированные рекомендации.
- **Администратор**: Полный доступ к данным через админ-панель, полный доступ к управлению контентом.

## 3. Модели данных
**Модель 1: SolarPanel** (Солнечная панель)
- Поля: название, производитель, мощность (Вт), КПД, цена, габариты, гарантия (лет)

**Модель 2: Region** (Регион)
- Поля: название, код региона, тариф день (руб/кВтч), тариф ночь, солнечные часы в год, координаты (широта/долгота)

**Модель 3: Calculation** (Расчет)
- Поля: пользователь (ForeignKey), регион (ForeignKey), панель (ForeignKey), количество панелей, месячное потребление, стоимость системы, годовая выработка, годовая экономия, срок окупаемости, сэкономлено CO2

## 4. Ключевой функционал
* Автоматический подбор оборудования на основе введенных параметров
* Расчет ROI с учетом региональных особенностей (тарифы, инсоляция)
* Генерация графиков окупаемости с помощью matplotlib
* Сохранение истории расчетов для авторизованных пользователей
* Сравнение разных конфигураций оборудования

## 5. Внешние интеграции / Аналитика
- **Внешние API**: NASA POWER API (данные по солнечной радиации), Mock API поставщиков (цены)
- **Аналитика**: pandas, matplotlib
